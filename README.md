# llm-ehr-backend

Backend for the BIOM9450 LLM-Enhanced Electronic Health Record (EHR) Clinical Decision Support System.
This service provides API endpoints and database logic for:

- User authentication

- Patient management

- EHR uploads

- LLM report generation

- Feedback + evaluation storage

Frontend repo: Next.js app (https://github.com/anishsinha2003/BIOM9450_MajorProject_Frontend)
